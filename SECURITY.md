# Security Policy

## Supported Versions

These Borg releases are currently supported with security updates.

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | :x: (beta)         |
| 1.4.x   | :white_check_mark: |
| 1.2.x   | :x: (no new releases, critical fixes may still be backported) |
| 1.1.x   | :x:                |
| < 1.1   | :x:                |
